Privacy & Data Control Gates

Grok Build loads configuration parameters from local TOML configurations and cloud policy headers. Adjust the settings below to see how the data pipelines respond dynamically.

Zero Data Retention (ZDR)

Forces the API to process data in-memory without persistent logs or model training storage. Configured in requirements.toml.

Coding Data Collection Opt-out

When checked, activates the is_data_collection_disabled() client check, bypassing trace uploads to the cloud.

Mixpanel Telemetry

Sends product usage and error analytics to telemetry services. Disabling this blocks outgoing telemetry hooks.
